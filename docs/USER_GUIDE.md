## インストール

1. GitHub Releases ページから最新の `.exe` インストーラーをダウンロードしてください
2. ダウンロードした `.exe` を実行してください
3. 画面の指示に従ってインストールを完了してください

> Windows SmartScreen の警告が出る場合は「詳細情報」→「実行」で進めてください。

## 初期設定（LLMモデルの接続）

アプリを起動したら、まずLLMの接続設定を行います。

### 設定画面を開く

1. 画面左下の **⚙（歯車）アイコン** をクリックしてください
2. 設定画面が表示されます

### モデル設定タブ

設定画面上部の **「モデル設定」** タブ（デフォルトで選択済み）に、4つの用途別モデル設定欄があります：

| 用途 | 説明 |
|------|------|
| チャット | キャラクターとの会話に使用されます |
| 記憶圧縮 | 会話の要約・長期記憶の生成に使用されます |
| 思考生成 | キャラクターの内部思考に使用されます |
| キャラクター生成 | System Promptの自動生成に使用されます |

### 各モデルの設定手順

1. **プロバイダー** ドロップダウンから選択してください：
   - `OpenAI` — GPT-4o 等
   - `Anthropic` — Claude 等
   - `Google` — Gemini 等
   - `OpenAI互換` — ローカルLLM（LM Studio, Ollama等）

2. **Base URL** — プロバイダーを選択すると自動でデフォルト値が入ります。OpenAI互換の場合は手動入力が必要です（例: `http://localhost:1234/v1`）

3. **API Key** — プロバイダーから取得したAPIキーを入力してください（目のアイコンで表示/非表示を切り替えられます）

4. **モデル名** — 使用するモデル名を入力してください（例: `gemini-2.5-flash`）
   - APIキーとBase URLを入力済みの状態で「**モデル一覧取得**」ボタンをクリックすると、APIから利用可能なモデルの一覧を取得してドロップダウンで選択できます

5. **Temperature** — スライダーで応答のランダム性を調整できます（0.0〜2.0）。0.0に設定するとランダム性が無くなり、同じ質問に対しては同じ回答が返されます。

6. **接続テスト** ボタンをクリックして接続を確認してください

7. すべての設定が完了したら、画面右上の **「保存」** ボタンをクリックしてください

> **ヒント**: 最初は「チャット」だけ設定すれば会話を始められます。他の用途は後から設定できます。

## Google Gemini 無料APIキーの取得方法

ローカルLLMの環境がない方でも、Google Gemini の無料APIを使ってすぐに始められます。

### 手順

1. **Google AI Studio にアクセス**
   - ブラウザで [https://aistudio.google.com/](https://aistudio.google.com/) を開いてください
   - Googleアカウントでログインしてください（お持ちでない場合は作成してください）

2. **APIキーを作成**
   - 左メニューまたはページ上部の「**Get API key**」をクリックしてください
   - 「**Create API key**」ボタンをクリックしてください
   - プロジェクトを選択（または新規作成）して「**Create API key in existing project**」をクリックしてください
   - 表示されたAPIキー（`AIza...` で始まる文字列）をコピーしてください

3. **KokoroChat に設定**
   - 設定画面 → モデル設定タブを開いてください
   - プロバイダー: **Google** を選択してください
   - モデル名: `gemini-2.5-flash`（無料枠で利用可能な高性能モデル）を入力してください
   - API Key: コピーしたキーを貼り付けてください
   - 「**接続テスト**」で確認してください → 「接続成功」と表示されればOKです
   - 「**保存**」をクリックしてください

### 無料枠で使えるモデル（2026年5月時点）

| モデル名 | 特徴 | 無料枠レート制限 |
|----------|------|-----------------|
| `gemini-2.5-flash` | 高速・低コスト、思考機能あり | 10 RPM / 250 RPD |
| `gemini-2.5-flash-lite` | 最軽量、大量リクエスト向け | 15 RPM / 1,000 RPD |
| `gemini-2.5-pro` | 最高性能、複雑な推論向け | 5 RPM / 100 RPD |
| `gemini-3-flash`（プレビュー） | 最新世代Flash、コーディング・エージェント性能向上 | 制限あり（プレビュー） |

> **注意**: Gemini 3 Pro は無料枠では利用できません（有料のみ）。  
> Gemini 3 Flash はプレビュー段階のため、レート制限が厳しめです。

- レート制限（1分あたりのリクエスト数）があるため、連続で大量に送信すると一時的にエラーになる場合があります
- 最新の無料枠情報は [Google AI for Developers 料金ページ](https://ai.google.dev/gemini-api/docs/pricing) でご確認ください

### 推奨モデル設定（全用途共通で使う場合）

4つの用途すべてに同じ設定を入れれば、1つのAPIキーで全機能が使えます：

- プロバイダー: **Google**
- モデル名: `gemini-2.5-flash`（バランス重視）または `gemini-3-flash`（最新性能重視）
- API Key: 取得したキー
- Temperature: 0.70（デフォルトのまま）

## キャラクターの作成

### キャラクター管理画面を開く

1. 画面左下の **👤（人型）アイコン** をクリックしてください
2. キャラクター管理画面が表示されます

### 新規作成

1. 画面右上の **「＋ 新規作成」** ボタンをクリックしてください
2. フォームが表示されます

### フォームの入力

| 項目 | 説明 |
|------|------|
| アバター | 左上の丸いエリアをクリックして画像を選択してください。トリミング画面で調整できます |
| 名前 | キャラクターの名前です（必須） |
| 概要 | 性格や背景の説明文です。System Prompt自動生成の材料になります |
| System Prompt | キャラクターの振る舞いを定義するプロンプトです |

### System Prompt の自動生成

1. 名前と概要を入力してください
2. System Prompt 欄の右上にある **「生成」** ボタンをクリックしてください
3. LLM（キャラクター生成用モデル）が概要を元にSystem Promptを自動作成します

### System Prompt の改善

1. 既にSystem Promptがある状態で **「改善」** ボタンをクリックしてください
2. どう改善したいかの案がある場合は、「改善の方向性」欄に指示を入力してから **「改善」** ボタンをクリックしてください（例: 「もっとツンデレ感を出して」）
3. LLMが現在のプロンプトを改良します

### TTS設定（任意）

フォーム下部の「🔊 TTS設定」セクションで、キャラクターごとの音声設定ができます：

1. 「TTSを有効にする」チェックボックスをONにしてください
2. プロバイダーを選択してください（VoicePeak または Irodori-TTS）
3. 各プロバイダー固有の設定を入力してください

### 保存

フォーム右下の **「保存」** ボタンをクリックしてください

## チャットの基本操作

### キャラクターを選択

1. サイドバー上部の **キャラクター選択ドロップダウン** をクリックしてください
2. 会話したいキャラクターを選択してください

### 新しいチャットを開始

- キャラクター選択後、メイン画面に **「新しいチャットを開始」** ボタンが表示されます
- クリックするとチャットセッションが作成されます

### メッセージの送信

1. 画面下部のテキスト入力欄にメッセージを入力してください
2. 送信方法（デフォルト: Enterキー、設定で変更可能です）
3. または右側の **送信ボタン（↑アイコン）** をクリックしてください

### 生成の停止

- AIが応答を生成中は、送信ボタンが **■（停止ボタン）** に変わります
- クリックすると生成を中断できます

### システムモード

入力欄左側の **ⓘ（情報アイコン）** をクリックすると「システムモード」に切り替わります：
- 状況説明やルールをキャラクターに伝えるためのモードです
- 通常の会話メッセージとは区別して送信されます
- もう一度クリックすると通常モードに戻ります

### メッセージの操作

各メッセージにマウスを乗せると操作ボタンが表示されます：
- **再生成** — AIの応答を再生成します
- **削除** — メッセージを削除します

## サイドバーの使い方

### 構成

サイドバーは上から順に以下のように構成されています：

1. **アプリ名（KokoroChat）** — クリックするとチャット画面に戻ります
2. **キャラクター選択** — ドロップダウンでキャラクターを切り替えられます
3. **タブ切替** — 「チャット」「思考」「記憶」の3タブがあります
4. **タブ内容** — 選択中のタブに応じたリストが表示されます
5. **下部ナビゲーション** — 💬チャット / 👤キャラクター管理 / ⚙設定

### チャットタブ

- 選択中キャラクターのチャットセッション一覧が表示されます
- クリックでセッションを切り替えられます
- 新規セッション作成ボタンもあります

### 思考タブ

- キャラクターの内部思考（思索）一覧が表示されます
- クリックで思考の詳細を閲覧できます

### 記憶タブ

- 会話から生成された長期記憶の一覧が表示されます
- クリックで記憶の内容を閲覧できます

## 思考・記憶の閲覧

思考、記憶が生成されるとその内容は同じキャラクターとの会話中参照されるようになります。（思考は一定期間で自動削除）  
これによりチャットを切り替えても以前の会話の内容を記憶したまま会話を続けられます。

### 思考（独自思考）

キャラクターはチャットとは独立して定期的に「思考」を生成します。

- サイドバーの **「思考」タブ** で一覧が表示されます
- 各思考をクリックすると内容を閲覧できます

### チャットヘッダーのコントロール

チャット画面上部の右側に以下のアイコンボタンがあります：

| アイコン | 機能 |
|----------|------|
| 🔊 スピーカー + スライダー | TTS音量を調整できます（TTS有効時のみ表示） |
| ✨ キラキラ | 記憶を手動で生成します |
| 🧠 脳 | 思考生成の一時停止/再開ができます（思考有効時のみ表示） |
| 💬 吹き出し | 自発的発話の一時停止/再開ができます（自発的発話有効時のみ表示） |

### 記憶の手動生成

1. チャットヘッダーの **✨（キラキラ）アイコン** をクリックしてください
2. 現在のチャット内容から記憶が生成されます

## ファイル添付

チャットにファイルを添付して、その内容について会話できます。

### 対応形式

- テキスト: `.txt`, `.md`, `.csv`
- ドキュメント: `.pdf`
- 画像: `.png`, `.jpg`, `.jpeg`, `.webp`

### 添付方法

**方法1: ボタンから**
1. 入力欄左側の **📎（クリップ）アイコン** をクリックしてください
2. ファイル選択ダイアログからファイルを選択してください

**方法2: ドラッグ＆ドロップ**
1. ファイルをチャット画面にドラッグしてください
2. 「ファイルをドロップして添付」のオーバーレイが表示されます
3. ドロップすると添付されます

添付されたファイルは入力欄の上にプレビュー表示され、×ボタンで削除できます。

## ツール管理（プラグイン）

キャラクターが会話中にツール（Web検索、計算など）を使えるようになります。LLMが必要に応じて自動的にツールを呼び出します。

### 設定画面を開く

1. ⚙設定画面 → **「ツール管理」タブ** を開いてください
2. 組み込みプラグインの一覧と設定が表示されます

### 組み込みプラグイン

| プラグイン | 機能 | 必要な設定 |
|-----------|------|-----------|
| Web検索 | キーワード検索とWebページ取得 | APIキー（Tavily or Brave） |
| 計算機 | 数式の計算 | なし |
| ファイル操作 | サンドボックス内のファイル読み書き | なし |

### Web検索の設定

1. **検索プロバイダ** を選択してください（Tavily または Brave Search）
2. **Tavily API Key** — [tavily.com](https://tavily.com) でアカウント作成後、APIキーを取得して入力してください
3. **Brave Search API Key** — [brave.com/search/api](https://brave.com/search/api/) でAPIキーを取得して入力してください
4. **許可ドメイン** — ページ取得（fetch_page）で許可するドメインを1行1つで入力してください（空の場合は全ドメインがブロックされます）
5. **「保存」** をクリックしてください

> **ヒント**: Tavily と Brave の両方のAPIキーを設定しておけば、プロバイダを切り替えてもキーが消えません。

### カスタムツールの追加

独自のツールを登録して、キャラクターに使わせることができます：

1. 「ツール管理」タブ下部の **「＋ カスタムツール追加」** をクリックしてください
2. 以下を入力してください：
   - **ツール名** — LLMが呼び出す際の識別名
   - **説明** — ツールの機能説明（LLMがいつ使うか判断する材料になります）
   - **種別** — HTTP Webhook または CLI コマンド
   - **エンドポイント/コマンド** — 呼び出し先のURLまたはコマンド
3. **「追加」** をクリックしてください

### ツールの有効/無効

各プラグインカードのトグルスイッチで、プラグインごとに有効/無効を切り替えられます。無効にしたプラグインのツールはLLMに提示されません。

### 会話中のツール使用

- キャラクターがツールを使用すると、チャット内に **ツール呼び出しインジケーター** が表示されます
- ツールの実行結果も折りたたみ表示で確認できます
- ツールの使用はLLMが自動判断するため、ユーザー側の操作は不要です

## TTS（音声合成）の設定

### グローバル設定

1. ⚙設定画面 → **「TTS」タブ** を開いてください
2. 「有効」チェックボックスでTTS全体のON/OFFを切り替えられます
3. VoicePeak実行ファイルパスを指定してください（未指定時はPATHから検索されます）
4. IrodoriTTSのベースURLを設定してください
5. 分割文字数・タイムアウトを必要に応じて調整してください

### キャラクター個別設定

キャラクター編集フォームの「🔊 TTS設定」セクションで個別に設定できます：

- **VoicePeak**: ナレーター名、感情パラメータ、速度、ピッチ
- **Irodori-TTS**: キャプションモード or 参照音声モード

---

## その他の設定

### 自発的発話（設定 → 「自発的発話」タブ）

- 有効/無効を切り替えられます
- 最小間隔（秒）を設定できます

### 思考（設定 → 「思考」タブ）

- 有効/無効を切り替えられます
- 生成間隔（分）を設定できます
- 自動削除閾値を設定できます（プリセットボタン: 無効/1時間/6時間/24時間/7日）

### 一般（設定 → 「一般」タブ）

- **記憶管理**: 圧縮閾値（何メッセージで記憶を圧縮するか）を設定できます
- **添付ファイル**: 最大ファイルサイズ（MB）を設定できます
- **送信キー**: Enter / Ctrl+Enter / Shift+Enter から選択できます

## トラブルシューティング

### 接続テストが失敗する

- APIキーが正しいかご確認ください
- Base URLが正しいかご確認ください（末尾に `/` が不要な場合があります）
- ネットワーク接続をご確認ください

### キャラクターが応答しない

- 設定画面で「チャット」用モデルが正しく設定されているかご確認ください
- 接続テストが成功するかご確認ください

### 自発的発話や思考が動かない

- 設定画面の各タブで「有効」になっているかご確認ください
- チャットヘッダーで一時停止されていないかご確認ください（アイコンが薄くなっていたら停止中です）
