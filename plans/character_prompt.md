# アプリの概要
このアプリは、ローカルLLMを活用した「AIキャラクターチャットデスクトップアプリケーション」です。
ユーザーは自分だけのAIキャラクターを作成し、会話を楽しめます。キャラクターは自発的に話し、独自に思考し、記憶を蓄積する高度な機能を持っています。

# アプリの画面構成と主要機能・操作方法

## 1. サイドバー（画面左側）
サイドバーの上部には3つのタブがあり、キャラクターの異なる側面にアクセスできます。
- **「チャット」タブ**: キャラクターとの通常の会話を行うメイン画面です。
- **「思考」タブ**: キャラクターの内部的な「思考（独り言や状況の考察）」を覗き見ることができるタブです。会話の裏でキャラクターが何を考えているかがわかります。
- **「記憶」タブ**: キャラクターが過去の会話から重要だと判断し、自動で要約・圧縮して保存した「長期記憶」の一覧を確認できます。

サイドバーの最下部には3つのアイコンボタンがあります。
- **💬 メッセージアイコン**: チャット画面に戻ります。
- **👥 ユーザーアイコン (キャラクター管理)**: 新しいキャラクターを作成したり、既存のキャラクターを切り替えたりできます。
- **⚙️ 設定アイコン**: アプリ全体の設定画面を開きます。

## 2. チャット画面とメッセージ入力
チャット画面の下部にある入力エリアには、便利な機能が備わっています。
- **通常の送信**: メッセージを入力し、Enterキーまたは「送信」ボタン（紙飛行機アイコン）で送信します。
- **📎 添付ファイル**: クリップアイコンをクリックすると、テキスト、PDF、画像などをチャットに添付してキャラクターに読み込ませることができます。
- **ℹ️ システムモード**: Infoアイコン（i）をクリックすると「システムモード」に切り替わります。入力欄の色が変わり、キャラクターの会話としてではなく「状況説明」や「ルールの追加」といったシステム指示を裏から送ることができます。
- **⏹️ 生成停止**: キャラクターが応答を生成している最中は、送信ボタンが四角い停止ボタンに変わります。これを押すことで応答を途中でキャンセルできます。

## 3. キャラクター作成機能
サイドバー下部のユーザーアイコンからキャラクター管理画面に行き、新しいキャラクターを作成できます。
- **自動生成**: キャラクターの名前と簡単な概要（設定）を入力するだけで、LLMが自動的に詳細な「システムプロンプト（性格や口調の指示）」を作成してくれます。
- もちろん、プロンプトは後から手動で自由に編集することも可能です。

## 4. 高度なAI機能
- **自発的発話**: ユーザーが何も入力しなくても、設定された間隔でキャラクターの方から自発的に話しかけてくる機能があります。
- **プラグイン (Tool Use)**: キャラクターは、必要に応じて自動的に拡張機能を使います。現在、計算機（Calculator）、ファイル操作（FileOps）、Web検索（WebSearch）などのツールに対応しており、複雑なタスクをこなすことができます。
- **用途別のモデル設定**: 設定画面から、役割ごとに異なるLLM（AIモデル）を割り当てることができます。「会話用」「記憶整理用」「思考用」「キャラクター生成用」に、それぞれ最適なモデルやAPIキーを使い分けることが可能です。

## 5. 音声合成 (TTS)
このアプリは「IrodoriTTS」や「VoicePeak」といった外部の音声合成ツールと連携できます。設定画面でエンドポイントを設定すれば、キャラクターの返答をフルボイスで読み上げさせることが可能です。
